# Tailored Intelligence Event Browser

A Kivy desktop application for browsing CrowdStrike Tailored Intelligence events — customer-environment-specific threat events generated by active tailored rules. It retrieves events and rules from the Falcon platform and presents them in a card-based feed with drill-down detail screens.

The Events tab displays a paginated, auto-refreshing card feed showing event type, headline, confidence (colour-coded), and timestamp. Tapping a card opens a full detail screen with the event description (automatically decoded from JSON/Lua-table body content when present), indicators, source rule cross-reference, and raw field dump.

The Rules tab lists active tailored intelligence rules with name, type, and enabled status. A date-range filter bar (24h / 7d / 30d / All) generates server-side FQL filters for narrowing the event window.

If credentials are absent or authentication fails, an error overlay is shown with instructions.

## Running the program

In order to run this demonstration, you will need access to CrowdStrike API keys with the following scopes:

| Service Collection | Scope |
| :---- | :---- |
| Tailored Intelligence | __READ__ |

### Execution syntax

This sample supports [Environment Authentication](https://falconpy.io/Usage/Authenticating-to-the-API.html#environment-authentication), meaning you can execute any of the command lines shown below without providing credentials if you have the values `FALCON_CLIENT_ID` and `FALCON_CLIENT_SECRET` defined in your environment.

> Launch with credentials passed via CLI flags.

```shell
python3 tailored_intel_browser.py -k $FALCON_CLIENT_ID -s $FALCON_CLIENT_SECRET
```

> Launch using environment variables (no flags needed).

```shell
python3 tailored_intel_browser.py
```
