# Tailored Intelligence examples
The examples within this folder focus on leveraging CrowdStrike's Tailored Intelligence API to browse customer-environment-specific threat events and rules.

- [Tailored Intelligence Event Browser](tailored_intel_browser) - Kivy desktop GUI for browsing tailored intelligence events and rules with a card-based feed, date-range filtering, and drill-down detail screens.
