# On-Demand Scan (ODS) examples
The examples within this folder focus on leveraging CrowdStrike's On-Demand Scan API to launch, monitor, and manage agent-side AV scans.

- [ODS Manager](ods_manager) - wxPython desktop GUI for creating on-demand scans, monitoring scan progress with auto-polling, inspecting per-host results, and cancelling scan jobs.
