# FileVantage examples
The examples within this folder focus on leveraging CrowdStrike's FileVantage API for file integrity monitoring.

- [FileVantage Change Monitor](filevantage_monitor) - PyQt6 desktop GUI for monitoring file integrity change events with auto-refresh, drill-down detail dialogs, suppress/purge actions, and CSV export.
