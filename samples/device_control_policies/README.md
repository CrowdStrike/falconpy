# Device Control Policies examples
The examples within this folder focus on leveraging CrowdStrike's Device Control Policies API to manage USB device control policies.

- [Device Control Policy Manager](device_control_policy_manager) - PySide6 desktop GUI for browsing, creating, editing, enabling/disabling, and deleting USB device control policies with host group assignment.
