# Zero Trust Assessment examples
The examples within this folder focus on leveraging CrowdStrike's Zero Trust Assessment API to visualise and audit ZTA scores across your fleet.

- [Zero Trust Assessment Score Viewer](zta_score_viewer) - Dear PyGui desktop GUI for viewing ZTA score distribution, sortable device table with hostname resolution, worst offenders panel, and audit report viewer.
